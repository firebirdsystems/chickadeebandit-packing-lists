# Packing Lists

Shared packing checklists for trips — build the list together, assign items to
people, start from a template (weekend / beach / camping / flight), and watch
the bags fill up.

- **Storage:** D1 (`lists`, `items`)
- **Access:** intentionally **no row policies** — packing is fully co-edited
  household data (kids check off their own socks). Nothing here needs
  protecting, and the UI implies no restriction.
- **AI:** read-only exports `lists`, `unpacked_items`.

## Develop

```bash
make install
make dev
make test
make build
```
